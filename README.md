Personal HTML Portfolio Website (Old Version)
Project Description
This is a personal portfolio website showcasing graphic design work, my learning process, and JavaScript effects. The website includes several static pages displaying my design projects, educational journey, and interactive features built using JavaScript.

Installation and Usage
Clone or download this project:

git clone https://github.com/TANANIS/TANA.git
After unzipping the files, open the folder, which contains HTML, CSS, and JS files.

Open the index.html file in a web browser to view the site.

To modify styles or effects, edit the corresponding CSS and JS files.

Features
Showcase Pages: Displays various design projects and learning milestones.
Interactive Effects: JavaScript-driven slide effects, animations, and more.
Simple Layout: Easy to navigate, focused on showcasing the portfolio.
Responsive Design: The site is optimized for both desktop and mobile devices.
Technologies and Tools
HTML: Used for building the structure of the website.
CSS: Applied for styling, layout, and design enhancements.
JavaScript: Implemented for interactive features and visual effects.
